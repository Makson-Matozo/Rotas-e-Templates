O que é o Solo Anime?

O Solo Anime é um blog feito com carinho para os fãs de animes clássicos. Aqui você encontra informações, personagens e trailers de grandes títulos como One Piece, Bleach, Naruto e Dragon Ball.

O projeto foi criado como parte de um trabalho escolar utilizando HTML, CSS e Bootstrap, para garantir uma experiência visual moderna e responsiva.

Objetivo do Projeto

Mostrar como é possível criar um site completo, bonito e organizado com tecnologias simples da web. Também buscamos despertar o interesse por animes em quem ainda não conhece e celebrar os clássicos que marcaram gerações.
